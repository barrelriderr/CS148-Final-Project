This is a site where people can create an account and enter the specs of their computer (laptop hardware not supported) and compare it to other's. All hardware data is stored in a database and is modified via content control tools accessed by the admin.

I built this site using an MVC design pattern, and light-weight PHP framework that this site runs on.
